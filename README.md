# Motion_Gesture_Remote_Car
Components - Transmitter:
- Arduino Nano 
- nrf24l01+ module
- nrf adapter 
- MPU6050 module 
- 7-12 V DC battery (In our case lipo 2s battery) 
- Breadboard  
- Double sided tape
- Jumper wires 

Components - Receiver:
- 4WD car kit (It has 4 TT DC gear motors with wheels)  
- Arduino Nano
- nrf24l01+ module
- nrf adapter 
- L298N driver module 
- 7-12 V DC battery (for car we will use lipo 3s battery) 
- Breadboard 
- Double sided tape
- Jumper wires
